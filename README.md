# Euler Sayısı: Matematiğin Doğal Sabiti

Bekir Ozan Demir için hazırlanmış, TED Konya Koleji 11. sınıf matematik projesi kapsamında Euler sayısı `e` üzerine kurulu etkileşimli eğitim sitesi.

## Çalıştırma

```bash
npm install
npm run dev
```

Tarayıcıda `http://localhost:3000` adresini açın.

## Üretim Derlemesi

```bash
npm run build
npm run start
```

## Seçilen Mimari

- Next.js App Router + TypeScript
- Tailwind CSS v4 ile token tabanlı tema sistemi
- Framer Motion ile ölçülü giriş animasyonları ve azaltılmış hareket desteği
- Recharts ile etkileşimli matematik görselleştirmeleri
- KaTeX ile temiz matematik gösterimi
- Lucide React ile tutarlı ikon dili
- `src/lib/content.ts` içinde düzenli Türkçe içerik ve kaynakça
- `src/lib/math.ts` içinde simülasyon hesapları

## Proje Güçlü Yönleri

- e sayısını yalnızca tanıtmıyor; limit, seri, türev, integral, doğal logaritma, bileşik faiz ve uygulamalar arasında anlamlı bir hikâye kuruyor.
- “Euler sayısı”, “Euler formülü”, `ln`, `π` ve Euler-Mascheroni sabiti arasındaki farkları açıkça ayırıyor.
- Etkileşimli laboratuvar bölümü; limit yakınsaması, bileşik faiz, Taylor serisi, grafik karşılaştırması, büyüme/bozunma ve mini quiz içeriyor.
- Kaynakça bölümü güvenilir kaynakların ne için kullanıldığını ve neden güvenilir olduklarını açıklıyor.
- Mobil, tablet ve masaüstü için duyarlı tasarım; klavye erişilebilirliği, odak durumları, renk kontrastı ve azaltılmış hareket desteği gözetildi.

## Etkileşimlerin Öğrenmeye Katkısı

- Limit aracı, `(1 + 1/n)^n` ifadesinin e'ye yaklaşmasını sayısal ve görsel olarak gösterir.
- Bileşik faiz simülatörü, kesikli bileşikleşmeden sürekli bileşikleşmeye geçişi karşılaştırır.
- Taylor serisi keşfi, sonsuz serilerin birkaç terimle nasıl güçlü yaklaşım verdiğini sezdirir.
- Grafik gezgini, `e^x` ve `ln(x)` fonksiyonlarının ters ilişkisini gösterir.
- Büyüme/bozunma modeli, `N(t)=N_0e^{kt}` fikrinin gerçek süreçlerde neden kullanıldığını anlatır.
- Mini quiz, temel kavramları hızlıca pekiştirir.
